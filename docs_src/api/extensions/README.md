# API Extensions

The extension methods in this library help bootstrap the services and wire up serialization rules for the widget registry.

## Extension Classes

- [ServiceCollectionExtensions](../BbQ.ChatWidgets.Extensions.ServiceCollectionExtensions.html) - Service registration and configuration extensions
- [WidgetActionHandlerExtensions](../BbQ.ChatWidgets.Extensions.WidgetActionHandlerExtensions.html) - Extensions for widget action handlers
- [WidgetRegistryExtensions](../BbQ.ChatWidgets.Extensions.WidgetRegistryExtensions.html) - Extensions for widget registry operations

For a complete list of extension methods, see the [API Reference TOC](../toc.html).
