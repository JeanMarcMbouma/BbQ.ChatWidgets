    Registers a custom widget instance with an optional custom type identifier.

    Declaration

    Parameters
    Type Name Description
    IWidgetRegistry registry

    The widget registry.
    ChatWidget instance

    The widget instance to register as a template.
    string typeIdOverride

    Optional custom type identifier. If not provided, the instance's Type is used.
    Examples
    var registry = services.GetRequiredService<IWidgetRegistry>();
var customWidget = new MyCustomWidget("Label", "action");
registry.RegisterCustom(customWidget);  // Uses default type from instance
registry.RegisterCustom(customWidget, "custom_type");  // Uses custom type ID
