    Remarks

    Agents are registered as keyed IAgent services via the AddAgent<TAgent>(IServiceCollection, string, ServiceLifetime) extension method. This implementation resolves agents on demand from the DI container, fully supporting all service lifetimes (singleton, scoped, transient) without requiring manual instance management.
