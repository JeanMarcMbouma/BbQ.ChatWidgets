    Remarks

    Cards are used to display featured items, products, recommendations, or other rich content. They support:

    • A title for the card
    • An optional description
    • An optional image URL
    • An action triggered by clicking the card or its button

    Example JSON: {"type":"card","label":"View Details","action":"view","title":"Product Name","description":"Description","imageUrl":"https://..."}
