Remarks

This provider supplies a basic set of AI-callable tools that extend the AI's capabilities beyond the standard widget tools. Currently provides a retry tool to allow the AI to explicitly request retry operations when needed.

Implements IAIToolsProvider and can be overridden by custom implementations via ToolProviderFactory.

This method returns a cached list of AI tools. On first call, it creates the tools and caches them for subsequent calls. The current implementation includes:

  • retry_tool: Instructs the AI to retry the previous logic or generation
