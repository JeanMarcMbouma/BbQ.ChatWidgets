## ?? USER SECRETS OVERVIEW

### What are User Secrets?
- .NET feature for secure local development
- Stores sensitive data outside project directory
- Never committed to git
- Easy to use

### Basic Commands
```bash
dotnet user-secrets init                    # Initialize
dotnet user-secrets set "key" "value"       # Store
dotnet user-secrets list                    # View (masked)
dotnet user-secrets remove "key"            # Remove
dotnet user-secrets clear                   # Clear all
```

### Stored Location
- Windows: `%APPDATA%\Microsoft\UserSecrets\{Id}\secrets.json`
- Mac/Linux: `~/.microsoft/usersecrets/{Id}/secrets.json`

### How It Works
1. Initialize secrets for project
2. Configuration builder loads secrets automatically
3. Access via configuration["key"]
4. Never committed to source control
