    Remarks

    Date picker widgets allow users to select a specific date within an optional date range. Common uses:

    • Appointment scheduling
    • Event date selection
    • Deadline specification
    • Birthday/date of birth selection

    Example JSON: {"type":"datepicker","label":"Select date","action":"pick_date","minDate":"2024-01-01","maxDate":"2024-12-31"}
