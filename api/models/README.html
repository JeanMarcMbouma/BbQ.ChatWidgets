    API Models

    Explore the widget, message, and thread models that flow through the service boundary. Each model has JSON converters and helper extensions in BbQ.ChatWidgets.Models.

    Widget Models

    Core Widgets

    • ChatWidget - Base class for all widgets
    • ButtonWidget - Clickable button widget
    • CardWidget - Card display widget
    • ImageWidget - Image display widget
    • ImageCollectionWidget - Collection of images
    • ProgressBarWidget - Progress indicator widget
    • ThemeSwitcherWidget - Theme switcher control

    Form Widgets

    • FormWidget - Container for form fields
    • FormField - Individual form field
    • FormAction - Form action definition
    • InputWidget - Text input field
    • TextAreaWidget - Multi-line text input
    • DropdownWidget - Dropdown select widget
    • MultiSelectWidget - Multi-select dropdown
    • SliderWidget - Numeric slider widget
    • ToggleWidget - Toggle/switch widget
    • FileUploadWidget - File upload widget
    • DatePickerWidget - Date picker widget

    Chat Models

    • ChatTurn - Represents a conversation turn
    • StreamChatTurn - Streaming conversation turn
    • ChatMessages - Message collection utilities

    Supporting Models

    • ImageItem - Image item for collections
    • WidgetTool - AI tool definition for widgets

    Utilities

    • ChatWidgetConverter - JSON converter for widgets
    • ChatWidgetExtensions - Extension methods for widgets
    • ChatMessageExtensions - Extension methods for messages
    • Serialization - Serialization utilities

    For a complete list of models, see the API Reference TOC.
