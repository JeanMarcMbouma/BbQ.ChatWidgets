# Widget Types

- `button`: clickable actions
- `card`: rich content with title/image
- `input`: single-line text input (must be placed inside a `form` widget)
- `textarea`: multi-line text input (must be placed inside a `form` widget)
- `dropdown`: pick an option (must be placed inside a `form` widget)
- `multiselect`: pick multiple options (must be placed inside a `form` widget)
- `slider`: select a number range (must be placed inside a `form` widget)
- `toggle`: flip a boolean (must be placed inside a `form` widget)
- `datepicker`: pick a date (must be placed inside a `form` widget)
- `fileupload`: upload files (must be placed inside a `form` widget)
- `form`: groups input-like widgets and submits a payload
- `progressbar`: display progress
- `themeswitcher`: choose a theme
