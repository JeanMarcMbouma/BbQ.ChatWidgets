    Remarks

    Sliders allow users to select a value within a defined range using visual feedback. Common uses:

    • Volume control
    • Price range selection
    • Rating scales
    • Confidence levels

    Example JSON: {"type":"slider","label":"Volume","action":"set_volume","min":0,"max":100,"step":5,"default":50}
