    Enum AgentEventType

    Defines the types of internal events that can be raised during agent processing.

    Namespace: BbQ.ChatWidgets.Agents
    Assembly: BbQ.ChatWidgets.dll
    Syntax
    public enum AgentEventType
    Remarks

    Consumers can subscribe to these events via IAgentEventHandler to observe what the agent pipeline is doing at any point in time — useful for building real-time "thinking" indicator widgets.

    Fields

    Name Description
    AgentCompleted

    Raised by MultiTurnAgentOrchestrator immediately after an agent in the pipeline has returned its result.
    AgentStarted

    Raised by MultiTurnAgentOrchestrator just before invoking an agent in the pipeline.
    Thinking

    Raised just before the AI model begins generating a response (i.e. the model is "thinking").
    ToolCallCompleted

    Raised when a tool invocation has finished (whether successfully or with an exception). The Message property contains the tool's name.
    ToolCallStarted

    Raised just before the AI model invokes a registered tool function. The Message property contains the tool's name.
    TriageCompleted

    Raised by a TriageAgent<TCategory> once it has classified the request and resolved the target agent.
    Triaging

    Raised by a TriageAgent<TCategory> when it begins classifying a request.
