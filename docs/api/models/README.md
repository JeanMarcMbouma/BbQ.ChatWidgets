# Models API

Documentation for data structures and types.

## Overview

Models define the data structures used throughout BbQ.ChatWidgets.

## Main Models

### ChatWidget

**Purpose**: Base class for all widget types

**Properties**:
- `Label` (string) - Display label
- `Action` (string) - Action identifier

**Derived Types**:
- `ButtonWidget` - Clickable button
- `CardWidget` - Rich content card
- `InputWidget` - Text input
- `DropdownWidget` - Selection list
- `SliderWidget` - Range control
- `ToggleWidget` - Boolean switch
- `FileUploadWidget` - File input

**Example**:
```csharp
var button = new ButtonWidget(
    Label: "Click Me",
    Action: "confirm"
);
```

### ChatTurn

**Purpose**: Represents a single message/response

**Properties**:
- `Role` (ChatRole) - User or Assistant
- `Content` (string) - Message content
- `Widgets` (ChatWidget[]) - Embedded widgets
- `ThreadId` (string) - Conversation ID

**Example**:
```csharp
var turn = new ChatTurn(
    Role: ChatRole.Assistant,
    Content: "Here's your response",
    Widgets: new[] { new ButtonWidget(...) },
    ThreadId: "thread-123"
);
```

### ThreadData

**Purpose**: Represents a conversation thread

**Properties**:
- `Id` (string) - Unique thread identifier
- `CreatedAt` (DateTime) - Creation timestamp
