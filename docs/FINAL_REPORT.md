2. Create file: `docs/guides/TOPIC.md`
3. Fill in sections (Overview, Steps, Examples)
4. Add link to `docs/INDEX.md`
5. Test all links work

### To Add Examples
1. Copy template from `docs/MANAGEMENT.md`
2. Create file: `docs/examples/TOPIC.md`

4. Add explanation
5. Add link to `docs/INDEX.md`

### To Add API Docs
