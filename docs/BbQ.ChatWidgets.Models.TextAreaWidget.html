    Remarks

    TextArea widgets are used when users need to enter multiple lines of text. They support:

    • Placeholder text shown when empty
    • Maximum length constraint
    • Minimum and maximum row constraints for sizing

    Example JSON: {"type":"textarea","label":"Enter message","action":"input","placeholder":"Your message here","maxLength":500,"rows":5}
