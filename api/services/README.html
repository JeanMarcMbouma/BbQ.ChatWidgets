    API Services

    This section lists the hosted services such as ChatWidgetService, DefaultThreadService, and SSE helpers that manage widget streaming.

    Core Services

    • ChatWidgetService - Main orchestrator for chat widget operations
    • DefaultThreadService - Default implementation of thread management
    • WidgetSseService - SSE streaming service for widgets

    Widget Tools & Providers

    • DefaultWidgetToolsProvider - Default widget tools provider
    • DefaultAIToolsProvider - Default AI tools provider
    • DefaultWidgetHintParser - Default widget hint parser
    • DefaultStreamPayloadValidator - Default stream payload validator

    Registries

    • WidgetRegistry - Central widget registry
    • DefaultWidgetActionRegistry - Default action registry
    • WidgetActionRegistry - Action registry implementation

    Action Handling

    • DefaultWidgetActionHandlerResolver - Default action handler resolver
    • WidgetActionHandlerExtensions - Action handler extensions
    • WidgetActionMetadata - Action metadata definition
