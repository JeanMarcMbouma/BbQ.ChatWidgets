    Remarks

    Use this widget when you want to show one image inline in a chat response. For a gallery of images, use ImageCollectionWidget.

    Example JSON: {"type":"image","label":"Open","action":"open_image","imageUrl":"https://...","alt":"Optional alt","width":320,"height":180}
