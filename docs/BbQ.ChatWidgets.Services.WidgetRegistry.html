    public sealed class WidgetRegistry : IWidgetRegistry
    Remarks

    This registry maintains a mapping of widget type identifiers to their template instances. Each registered instance serves as the template for generating AI tools via WidgetTool.

    The registry is initialized with all built-in widget types and can be extended with custom instances at runtime.

    Key principle: Register actual widget instances, not types. The instance becomes the template used to generate AI tools.
